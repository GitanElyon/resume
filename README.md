# Overview

This repo holds my resume, CV and other professional doucments.

I plan to continously update these as my carreer progresses. 

You can view the live version here: [gitanelyon.github.io/resume](https://GitanElyon.github.io/resume).

This would allow anyone to access my resume online with real time updates.

# Why

For the longest time, I've been using google drawings to create my resumes.

Dispite being extremly niche, and somewhat limiting, it gave me good foundation I need to create a very basic resume, but recently, as I make more and more changes to my resume, its getting increasely difficult to manage.

To solve this, I switched to creating my resumes in HTML, which offer a bunch of benefits:
- Version control: I can see and even rollback to previous versions.
- Creative freedom: Canva, Figma, Google, etc, are a little limiting, but this gives me the creative freedom I want as a developer.
- Internet accessible: Anyone can now see my resume online and see what I have been up to recently.

# Usage

Simply visit the page where this is being hosted.

You can either view it in your browser or save it to your computer as a PDF.

Because it is an HTML page, I will be able to customize the CSS rules change how the PDF renders the HTML.
